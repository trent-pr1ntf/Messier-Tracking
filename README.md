# Messier-Tracking
Collection of CSV and XLSX files to track Messier objects observed for astrophotography. 

# Files
[Messier.csv](https://github.com/trent-pr1ntf/Messier-Tracking/blob/edb883b0d1d9cac13625500cf6e8d689747a494e/Messier.csv)
Starting point CSV file downloaded from the internet, may need to clean up. Compiled by Darrell W. Green. 

[Messier-Objects.csv](https://github.com/trent-pr1ntf/Messier-Tracking/blob/edb883b0d1d9cac13625500cf6e8d689747a494e/Messier-Objects.csv)
Cleaned up CSV file with all Messier objects. 
Fields: Name	NGC	Constellation	Class	Right ascension	Declination	Magnitude	Angular size	Burnham	Remarks

[Messier-Galaxy-Tracking.xlsx](https://github.com/trent-pr1ntf/Messier-Tracking/blob/edb883b0d1d9cac13625500cf6e8d689747a494e/Messier-Galaxy-Tracking.xlsx)
Excel Spreadsheet I personally use to track integration time, as well as meridian date gathered from a sky atlas book, and other notes. 
Fields: Messier	NGC	Name	Constellation	Class	Type	Right ascension	Declination	9pm Meridian	Integration Time	Part of Mosaic	Notes
